Slowder: The Slow Image Loader
A hackathon project for NYU ITP's Stupid Hackathon

By Sylvia Naples and Jason Sigal

Since the theme of the Hackathon was to make things with no value, we decided to write a program that would load images slowly. The image loads with a fractal space-filling curve called the Hilbert curve. Because of it's fractal nature, the percent loaded will get closer and closer to 100; however, to load the image completely would take infinite time.
I wrote the program for the Hilbert curve in JavaScript using nothing but loops and recursion.


To load your own image into slowder, check out our Slowder page. (Currently only works in Google Chrome).
